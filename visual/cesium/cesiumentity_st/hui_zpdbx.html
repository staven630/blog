# PolygonGraphics (opens new window)

名称 类型 是否必填 默认值 描述
show Property (opens new window) | boolean <可选> true 用于指定多边形的可见性
hierarchy Property (opens new window) | PolygonHierarchy (opens new window) <可选> 该属性指定 PolygonHierarchy。
height Property (opens new window) | number <可选> 0 用于指定多边形相对于椭球面的高度。
heightReference Property (opens new window) | HeightReference (opens new window) <可选> HeightReference.NONE 指定高度相对于什么的属性。
extrudedHeight Property (opens new window) | number <可选> 用于指定多边形的凸出面相对于椭球面的高度。
extrudedHeightReference Property (opens new window) | HeightReference (opens new window) <可选> HeightReference.NONE 指定了 extrudedHeight 相对于什么。
stRotation Property (opens new window) | number <可选> 0.0 用于指定多边形纹理从北方逆时针旋转。
granularity Property (opens new window) | number <可选> Cesium.Math.RADIANS_PER_DEGREE 指定每个纬度和经度点之间的角距离。
fill Property (opens new window) | boolean <可选> true 用于指定多边形是否被所提供的材料填充。
material MaterialProperty (opens new window) | Color (opens new window) <可选> Color.WHITE 用于指定用于填充多边形的材料。
outline Property (opens new window) | boolean <可选> false 用于指定是否勾勒出多边形。
outlineColor Property (opens new window) | Color (opens new window) <可选> Color.BLACK 指定 Color 轮廓的属性。
outlineWidth Property (opens new window) | number <可选> 1.0 指定轮廓宽度的数字属性。
perPositionHeight Property (opens new window) | boolean <可选> false 指定是否使用每个位置的高度。
closeTop Boolean | boolean <可选> true 如果为 false，则将挤出的多边形顶部留空。
closeBottom Boolean | boolean <可选> true 如果为 false，则将挤出的多边形的底部保留为开放状态。
arcType Property (opens new window) | ArcType (opens new window) <可选> ArcType.GEODESIC 多边形边缘必须遵循的线型。
shadows Property (opens new window) | ShadowMode (opens new window) <可选> ShadowMode.DISABLED 用于指定该多边形是投射还是接收来自光源的阴影。
distanceDisplayCondition Property (opens new window) | DistanceDisplayCondition (opens new window) <可选> 用于指定要在距相机的距离处显示此多边形。
classificationType Property (opens new window) | ClassificationType (opens new window) <可选> ClassificationType.BOTH 用于指定此多边形在地面上时是对地形，3D 瓷砖还是对两者进行分类。
zIndex ConstantProperty (opens new window) | number <可选> 0 指定用于排序地面几何图形的 zIndex 的属性。仅当多边形为常量且未指定高度或拉伸高度时才有效。
